# 🇨🇳 Vercel + .cn 域名部署指南

既然您选择了 **Vercel** 部署并打算申请 **.cn** 域名,以下是为您量身定制的步骤。

## 第一步：申请 `.cn` 域名

1.  **选择注册商**: 推荐 **阿里云** 或 **腾讯云**。
2.  **搜索并购买**: 搜索 `chlj.cn`,如果可用则购买。
3.  **实名认证 (核心)**:
    - 购买后,您必须根据注册商的指引提交身份证信息进行实名认证。
    - **没有实名认证的 .cn 域名无法进行 DNS 解析**,这意味着您无法将其指向 Vercel。
    - 认证通常需要 1 个工作日。

## 第二步：Vercel 部署

1.  **推送代码**: 确保代码已上传到 GitHub。
2.  **导入项目**: 在 [Vercel](https://vercel.com) 导入您的仓库。
3.  **配置环境变量**:
    - 在 Vercel 项目设置中添加：`VITE_ZHIPU_API_KEY` = `您的智谱 API Key`。
4.  **部署**: 点击 Deploy。

## 第三步：绑定域名 `www.chlj.cn`

1.  **Vercel 设置**:

    - 在项目面板点击 `Settings` -> `Domains`。
    - 输入 `www.chlj.cn` 并点击 `Add`。
    - Vercel 会提示需要配置 DNS。

2.  **配置解析 (在阿里云/腾讯云后台)**:
    - **记录类型**: `CNAME`
    - **主机记录**: `www`
    - **记录值**: `cname-china.vercel-dns.com` (这是 Vercel 专门为中国优化过的加速地址)。
    - **TTL**: 默认即可。

## 常见疑问解答

- **需要 ICP 备案吗？**

  - **不需要**。因为 Vercel 的服务器在海外（通常是新加坡、香港或美国）,不属于国内节点。根据规定,只有放在国内服务器上的网站才必须显示备案号。
  - **但是**: 既然用的是 `.cn` 域名,**实名认证**是躲不掉的。

- **国内访问速度如何？**
  - 使用 Vercel 提供的 `cname-china.vercel-dns.com` 解析后,国内访问通常比较流畅。
